## Prompt Engineering

GCES Framework: 

- **Goal**: State the task and desired outcome.
- **Context**: Provide the background the AI needs to respond correctly. 
- **Expectations**: Add constraints and priorities
- **Sources**: Specify what information the AI should use in its response. 

GCES Framework applied to Endura

- **Goal**: next day training plan or weekly forecast plan output. 
- **Context**: Providing the model with past 7-10 days of training workouts, goal metrics, race day timeframe, similar past situations.
- **Expectations**- Should only consider the user data given and make predictions about what is best, only output data in this format, match this json schema. 
- **Sources**: Basically take certain data from the tables that matches the query.
